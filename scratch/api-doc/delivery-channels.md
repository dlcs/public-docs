# iiif-av policyData: more transcoding options not yet available

The `policyData` for `iiif-av` currently only supports a fixed set of preset strings. The following is noted in the page as an Aside but is not yet implemented:

> In a future version of the API you will have more options for specifying specific transcoding options in the `policyData`, and the members of the JSON array will be JSON objects carrying that data.

# iiif-av: HLS adaptive bitrate not yet available

> In a future version of the API, the platform will support adaptive bitrate outputs in HLS format.
